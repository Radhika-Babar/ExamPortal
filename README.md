# 🎓 Exam Portal (Full Stack)

A full-stack web application for conducting and managing online examinations.  
Built with modern technologies, this platform supports secure authentication, role-based access, and real-time exam workflows for students and faculty.

---

## 🚀 Features

### 🔐 Authentication
- User registration & login
- JWT-based authentication
- Role-based access (Student / Faculty)

### 👨‍🎓 Student Features
- View available exams
- Attempt tests
- Submit answers
- View results

### 👩‍🏫 Faculty Features
- Create and manage exams
- Add questions (MCQ-based)
- Monitor student submissions
- View results & analytics

### 📊 Exam System
- Timer-based exams
- Auto submission
- Question navigation panel
- Result calculation

---

## 🛠️ Tech Stack

### Frontend
- React (Vite)
- Tailwind CSS
- Axios
- React Router

### Backend
- Node.js
- Express.js
- MongoDB
- JWT Authentication
